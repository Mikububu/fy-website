WhatsApp Onboarding Setup

WhatsApp Business Account ID

Enter your WhatsApp Business Account ID. You can find this in your Meta Business Suite under WhatsApp Manager → Account Tools → Phone Numbers.

This is a 15-digit number identifying your business account.

Phone Number ID

Enter the Phone Number ID associated with your WhatsApp Business number. This is different from the actual phone number.

Found in Meta Business Suite under the phone number settings.

Permanent Access Token

Enter your permanent access token for the WhatsApp Business API. This is generated in the Meta Developer Portal.

Security Note: This token provides full access to your WhatsApp Business account. Keep it secure and never share it publicly.

Generate this in Meta Developer Portal → Your App → WhatsApp → API Setup.

Webhook Verify Token

Create a verify token for your webhook. This is a secret string that Meta will use to verify your webhook endpoint.

Create any secure string. You'll need to enter the same token in Meta's webhook configuration.

Webhook URL

Your webhook URL will be where Meta sends incoming WhatsApp messages. For this site, the webhook will be:

This URL is pre-configured for your Netlify deployment. Copy this to Meta's webhook settings.

Hosting Environment

Confirm your preferred hosting environment for the webhook.

Netlify is pre-configured for this site. Other options require additional setup.

PayPal Client ID

Enter your PayPal API Client ID for processing credit card and PayPal payments.

Found in PayPal Developer Dashboard → Apps & Credentials.

PayPal Client Secret

Enter your PayPal API Client Secret.

Security Note: This secret provides access to your PayPal account. Keep it secure and never share it publicly.

Found alongside the Client ID in PayPal Developer Dashboard.

WhatsApp Phone Number

Enter your WhatsApp Business phone number in international format (without + or spaces).

This is the number users will message. Format: country code + number (no spaces or symbols).

WhatsApp Broadcast List / Group

Enter the ID or link for your WhatsApp broadcast list or group where interested users will be added.

Optional: Leave blank if you'll manage subscriptions manually.
